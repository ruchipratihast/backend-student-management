# Student Management API
### A secure .NET backend built with EF Core and PostgreSQL. It manages student-course relationships using a structured Service Layer and DTO mapping.

#### Key Features
**Database:** Relational schema with Foreign Key constraints in PostgreSQL.

**Security:** JWT Authentication and Authorization for protected endpoints.

**Validation:** FluentValidation for clean, reliable input handling.

**Architecture:** Clean separation of concerns using the Service/Repository pattern.

**Deployment:** Fully containerized with Docker Compose.

This project demonstrates proficiency in C#, relational data integrity, and modern API security practices.
